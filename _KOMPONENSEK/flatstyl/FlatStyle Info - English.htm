Installation of the componentes

Delphi 5 Open the file flatstyle_d5.dpk and click to install.
Delphi 4 Open the file flatstyle_d4.dpk and click to install.
Delphi 3 Open the file flatstyle_d3.dpk and click to install.
Delphi 2 Add the file TFlatRegister.pas to the components.
C++ Builder 1 Add the file TFlatRegister.pas to the components.
C++ Builder 4 Open the file flatstyle_cb4.bpk and click to install.

Help

Please see "FlatStyle.chm" for the description of the FlatStyle-components.

If you can't open the helpfile you must install the HTML-Help. See the following page to download the file hhupd.exe in the language you need:

MS HTML Help Update - Downloadpage

After install you can read the helpfile.
