Installieren der Komponenten

Delphi 5 Datei flatstyle_d5.dpk öffnen und auf installieren klicken.
Delphi 4 Datei flatstyle_d4.dpk öffnen und auf installieren klicken.
Delphi 3 Datei flatstyle_d3.dpk öffnen und auf installieren klicken.
Delphi 2 Datei TFlatRegister.pas zu Komponenten hinzufügen.
C++ Builder 1 Datei TFlatRegister.pas zu Komponenten hinzufügen.
C++ Builder 4 Datei flatstyle_cb4.bpk öffnen und auf installieren klicken.

Hilfe

Eine Beschreibung der FlatStyle-Komponenten finden Sie in der Hilfedatei "FlatStyle.chm".

Sollten Sie die Hilfedatei nicht öffnen können so installieren Sie bitte die Datei hhupd.exe in der von Ihnen gewünschten Sprache. Die Datei befindet sich im Internet unter:

MS HTML Help Update - Downloadpage

Nach der Installation sollten Sie auf die Hilfedatei zugreifen können.
